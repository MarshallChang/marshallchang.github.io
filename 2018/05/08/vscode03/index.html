Visual Studio Code Extensions 发布到Marketplace

Posted on 2018-05-08 | Edited on 2018-07-05 | In Technology

插件编写完成后,使用npm 安装vsce npm install -g vsce vsce --help查看帮助命令 登陆Visual Studio 进行一个团队的注册 创建 Personal Access Token 并进行保存 在插件项目中使用 vsce package 进行打包动作,生成 .vsix文件,在应用市场上传的时候只上传该文件即可. 该插件项目需要在package.json 中声明

在创建项目之初就将项目上传到github 是一个明智之举, 另外在进行打包动作时,README.md文件中也应该进行自己插件的一个描述,否则会打包检查失败造成无法打包. 之后即可在应用市场搜索下载 package.json文件中 engines指定了该插件最低兼容的软件版本,如无法安装请安装最新版本的vscode.
